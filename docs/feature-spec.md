# 功能规格说明

## 第一阶段 MVP 功能

### 1. 文件打开

- 支持打开 `.md`、`.markdown`、`.txt` 文件
- 通过系统文件选择对话框选择文件
- 通过 Tauri 后端读取文件内容
- 文件大小限制：20MB
- 错误处理：文件不存在、权限不足、编码异常

### 2. Markdown 渲染

- 支持标题 H1-H6
- 支持有序/无序列表
- 支持引用块
- 支持代码块（含语法高亮）
- 支持表格
- 支持链接（外部链接通过系统浏览器打开）
- 支持图片（本地相对路径图片）
- HTML 安全过滤（DOMPurify）

### 3. 主题切换

- 亮色主题
- 暗色主题
- 跟随系统主题
- 配置持久化

### 4. 字体设置

- 字号调整
- 行高调整
- 配置持久化

### 5. 内容宽度

- 窄屏阅读
- 标准阅读
- 宽屏阅读
- 配置持久化

### 6. 最近文件

- 打开文件后自动记录
- 最多保留 20 个
- 自动去重
- 点击重新打开
- 文件不存在时可移除
- 配置持久化

### 7. 启动恢复

- 恢复上次主题、字号、行高、宽度设置
- 可选自动打开上次文件

## 第二阶段功能（后续）

3. 拖拽文件到窗口打开
4. 代码块复制按钮
5. 图片相对路径显示
6. 沉浸式阅读模式
7. 页面宽度切换（窄/标准/宽）
8. 字体配置（字号、行高、字体族）

5. 记住上次打开的目录
6. 文件变更手动刷新
